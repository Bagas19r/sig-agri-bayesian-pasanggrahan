# sig-agri-bayesian-pasanggrahan
Dashboard SIG-Agri Eksekutif berbasis R Shiny untuk pemantauan kesehatan vegetasi sawah di Desa Pasanggrahan, Subang. Mengintegrasikan Data Fusion Satelit Sentinel-1 (Radar SAR) dan Sentinel-2 (Optik) guna mengatasi tutupan awan menggunakan pendekatan Standardisasi Spasiotemporal Bayesian. Dilengkapi otomasi pipeline data tahunan via GitHub Actions
